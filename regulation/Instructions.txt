This script, dep1_dlpfc.m, is the core code for performing computational brain network regulation (virtual perturbation) analysis as described in the associated publication. It calculates changes in whole-brain functional connectivity and derived network metrics in response to a model-based stimulation of a specified brain region. The regulation strategies targeting other brain regions (e.g., global, vmPFC, APCTh, IFHPT) for the two subtypes are not provided as separate scripts; to replicate those analyses, users can modify the relevant target_region parameter within this script accordingly, as the underlying computational framework remains identical and only the stimulation target needs to be changed to obtain the corresponding regulatory outcomes.
